Plataforma E-Ventos

Este fue mi proyecto más ambicioso. La plataforma le permite a los usuarios navegar exposiciones virtuales desde cualquier parte del mundo. Se puede navegar por los distintos halls y sus stands y atender conferencias y conciertos. También contiene un chat y una conexión a un sitio de networking.

Las imágenes son de Spannabis Virtual Edition, el primer evento llevado a cabo en esta plataforma, con +4000 invitados.

AgroRRHH

Esta plataforma es una bolsa de trabajo para el sector agropecuario. El sitio tiene dos tipos de usuarios, diferentes tipos de suscripciones pagas y un panel de administrador. Para poder desarrollarla aprendí a usar la tecnología back-end PHP para conectar el front-end de la página con una base de datos MySQL. Fueron usados HTML, CSS, JS, JQuery, PHP y MySQL para completar el desarrollo.

AsesorRural

En esta plataforma, usuarios registrados pueden publicar promociones de cursos y algunos usuarios específicos pueden publicar videos. Incluye un panel de administrador que permite verificar usuarios entre otras funcionalidades. Para el desarrollo usé HTML, CSS, JS, JQuery, PHP y MySQL. La página no está en uso todavía.
